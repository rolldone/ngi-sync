# Installation

Like npm as usual. just :

```bash
npm install ngi-sync -g
```

or clone from github

```bash
git clone git@github.com:rolldone/ngi-sync.git ngi-sync
```

then :

```bash
npm install copyfiles -g
```

```bash
cd ngi-sync && yarn install
```

```bash
npm run build && npm link --force
```
